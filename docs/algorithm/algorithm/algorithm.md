# 数据结构与算法

## 数据结构

- 一维  
  基础：数组 array、链表 link list  
  高级：栈 stack、队 queue
- 二维  
  基础：树 tree、图 graph  
  高级：二叉搜索树 binary search tree、堆 heap, 并查集 disjoint set, 字典树
- 特殊  
  位运算 bitwise, 布隆过滤器 Bloom Filter  
  LRU Cache
